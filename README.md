# Правила сервера WILD

! Важно: игра разрешена только через WILD Launcher.

## Основные правила

- Уважайте игроков и администрацию.
- Запрещены читы, макросы и обходы античита.
- Не используйте баги игры для преимущества.

? Подсказка: строки с `!` становятся красными предупреждениями, строки с `?` - синими подсказками.

> Так можно выделить важную заметку зелёным блоком.

---

### Как писать красиво

Используйте **жирный текст**, *акцент*, списки и заголовки `#`, `##`, `###`.
